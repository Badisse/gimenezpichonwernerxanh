## Get Started

Cloner le repertoire
```sh
$ git clone https://github.com/CharlieXanh/gimenezpichonwernerxanh.git
```

## Build le projet

```sh
$ cd gimenezpichonwernerxanh
$ mkdir build
$ cd build
$ cmake ..
$ make -j
```

## Affichage d'un message

```sh
$ ./bin/client hello
```
